# Aesthete Recipes

This is a wordpress theme that works well for the static site generation of the
recipes.hoza.us website. It does things like

- Removes RSS and other links in the "head" element.
- Removes speculation rules and emoji support for anchient browsers.
- Removes "Comments Off" notices on the posts.
- Etc.
